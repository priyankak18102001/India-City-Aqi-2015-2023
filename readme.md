# India Air Quality Dashboard (Streamlit)

This project is an interactive dashboard built using Streamlit to analyze air quality data of Indian cities.
It shows AQI trends, pollutant levels, city comparisons, and different visualizations to understand pollution patterns.

## Features

* KPI cards (Total records, Total cities, Average AQI)
* AQI trend over time
* AQI distribution histogram
* Average AQI by city
* Top polluted cities
* Pollutant trends (PM2.5, PM10, NO2, SO2, CO, O3)
* Correlation table
* Box plot and monthly trends
* Interactive dashboard

## Tools and Technologies

* Python
* Streamlit
* Pandas
* Matplotlib

## How to Run Locally

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the application:

```
streamlit run app.py
```

3. Open the browser and the dashboard will appear.
